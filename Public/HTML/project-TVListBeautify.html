TVListBeautify

Privat habe ich eine einfache Flutter-App entwickelt, die eine Liste von Serien und Filmen in eine ansprechende, visuell filterbare Liste umwandelt. Ursprünglich war auch geplant, dass Benutzer die Serien bewerten können, jedoch wurde diese Funktion nicht vollständig umgesetzt. Ich hatte nie wirklich vor, diese App zu veröffentlichen, weshalb der API-Key direkt in der App hinterlegt war. Diese ist natürlich nicht mehr gültig.

Verwendete Technologien

  • Flutter
  • Dart
  • Git
  • GitHub
  • Miro

Details

Dieses Projekt entstand aus persönlichem Interesse. Ich hatte bereits eine „Watchlist“ in meiner Notizen-App, die ich visuell besser gestalten wollte.
